📚 仓库简介

本仓库提供了一套经过严格清洗的高质量中文基础词库资源，旨在为中文自然语言处理（NLP）提供坚实的数据支撑。所有数据均已去重，开箱即用。

🗂️ 核心数据一览

中文字表.txt (汉字表)

收录字数：20,933 个独立汉字。

覆盖范围：全面收录简体中文与繁体中文，覆盖范围远超常用字库（约等于 GBK 全部汉字及部分扩展生僻字），适合用于字符级编码、生僻字校验及 OCR 场景。

中文词表.txt (词库)

收录词条：12,044,942 条（约 1204万+）。

数据特征：无重复、高质量，词条长度覆盖短词（二字词）、中词（三至四字）及长短语。

特色亮点：包含大量真实人名数据，极大提升命名实体识别（NER）的泛化能力。

💡 适用场景

中文分词（Word Segmentation）基准词库

语序验证码还原

敏感词过滤、搜索引擎倒排索引构建

汉字字形/字频统计与分析
